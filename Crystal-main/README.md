# Crystal
Client and application management system

### Steps to run this project:
1. `git clone ...`
2. `npm i install`
3. create a local .env file and insert mongodb key
4. `nodemon Crystal.js`

### Steps to merge 
1. $git merge origin/branchName
2. resolve the conflict
3. $git add *
4. $git push
